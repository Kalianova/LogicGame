LogicGame


Инструкция по сборке

1.	Скачать архивом 
2.	Извлечь папку LogicGame-master из архива
3.	Для сборки решения должен быть установлен CMake версии 3.10.0 и выше, а также Qt версии 5.12.5 и выше и генератор
4.	Открыть CMake. В верхнем поле указать путь к папке LogicGame-master, в нижнем путь к директории сборки
5.	Нажать Configure. Выбрать ваш генератор. Нажать Finish
6.	Если папка Qt не находится в корне диска C:\\, то в поле Qt5_DIR нужно будет указать путь к папке по подобию:
(папка в которой находится Qt)\Qt\5.12.5\(выбранный генератор)\lib\cmake\Qt5
7.	Нажать Configure. Нажать Generate
8.	Собрать проект с помощью выбранного генератора
9.	Запустить готовое решение
